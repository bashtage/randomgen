randomgen.xoroshiro128.Xoroshiro128.seed¶

Xoroshiro128.seed(seed=None)¶

Seed the generator.

This method is called at initialized. It can be called again to re-seed the generator.

Parameters

seed ({int, ndarray}, optional) – Seed for PRNG. Can be a single 64 biy unsigned integer or an array of 64 bit unsigned integers.

Raises

ValueError – If seed values are out of range for the PRNG.

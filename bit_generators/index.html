    Bit GeneratorsÂ¶

    The random values produced by numpy.random.Generator (and also Generator) are produced by a bit generator. These bit generators do not directly provide random numbers and only contain methods used for seeding, getting or setting the state, jumping or advancing the state, and for accessing low-level wrappers for consumption by code that can efficiently access the functions provided, e.g., numba.
